# License Options

No final license has been selected.

## MIT

Short, permissive, common for small tools. Users can reuse and modify the code with minimal conditions.

## Apache-2.0

Permissive like MIT, with explicit patent language. Useful when you want a more complete legal framework.

## GPL-3.0

Copyleft license. Modified redistributions must generally remain GPL-compatible and publish source.

## Recommendation

MIT is the simplest fit for a small PowerShell utility intended for broad personal use, unless the owner specifically wants patent language or copyleft obligations.
